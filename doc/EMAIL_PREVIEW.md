## 🎨 Design Features

All emails include:

✅ **Purple gradient header** - Professional, modern look
✅ **Responsive layout** - Works on mobile and desktop
✅ **Clear hierarchy** - Easy to scan and read
✅ **Action buttons** - Prominent CTAs where needed
✅ **Info/warning boxes** - Important information stands out
✅ **Security focused** - Clear warnings and instructions
✅ **Brand consistent** - Uses PROJECT_NAME throughout
✅ **Professional footer** - Legal info and unsubscribe

---

## 📱 Mobile Responsive

All templates automatically adjust for mobile:

- Font sizes scale down
- Buttons become full-width
- OTP codes remain readable
- Layout stays clean

---

## 🎯 Customization

To customize colors, fonts, or layout:

1. Edit `src/services/email/email.templates.ts`
2. Modify the `baseTemplate` function
3. Change CSS in the `<style>` section

---

**Preview in Browser:**
To see actual HTML rendering, you can:

1. Copy template code from `email.templates.ts`
2. Save as `.html` file
3. Open in browser
4. Or use online HTML viewers

**All templates are production-ready and automatically sent by your auth system!** 🚀
